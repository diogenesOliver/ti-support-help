<h1>Support Help Project</h1>

O projeto consiste basicamente no desenvolvimento de uma API Rest que tem como regra de negócio principal a criação de uma aplicação para auxilio na abertura de chamados técnicos para o time de T.I das empresas. De forma exemplificada podemos supor que um colaborador está enfrentando um problema técnico no seu computador ou software interno da empresa, dessa forma ele pode abrir um chamado dentro do hub para pedir auxilio para o time de T.I.

O intuito é simplificar e gameficar a abertura de chamados para que a comunicação e a relação entre colaborador/T.I seja feita de forma dinâmica e rápida.
